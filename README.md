## Section 1: Software and platform section
In this project, we used Python in Google Colab as our coding language. To complete exploratory data analysis pandas, numpy, seaborn, and matplotlib were utilzied. Specificially, we used the Python package yfinance for data related to the S&P 500. Both Mac and Windows were used.
